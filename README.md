## Công nghệ sử dụng

- **Frontend:**
  - HTML/CSS
  - JavaScript
  - Bootstrap 5 (UI Framework)

- **Backend:**
  - ASP.NET Core (C#)
  - Entity Framework Core
  - Microsoft SQL Server
